# FlareSpark

FlareSpark is a AI agent which embeddeds with zed using it's custom ACP agent which uses AI models provided by [Groq](https://console.groq.com).
it is my personal Coding Asisstant which helps me solving tiny bugs, resolve errors and provide small code corrections. To use this Agent you'll need a Groq API key which you can get from [Groq console](https://console.groq.com/keys).
I am currently testing and improving this ACP agent.
I would love if you want to contribute.
Check out [how it works](howitworks.md) to understand the workflow and get started.
feel free to submit PRs and open issues.
